# 돔, 봄

## DOM (HTML)

### `window` 객체

>
> 현재 JS 코드가 작동하고 있는 `창` (탭) 객체
> 
> 전역으로 접근가능하며, 속성값들은 `window` 키워드를 생략한 형태로 사용 가능
>

https://developer.mozilla.org/ko/docs/Web/API/Window

#### `document` 객체

> `window.document`
>
> 현재 `창` 의 `문서` (`DOM`)
>

- getElementById
- getElementsByClassName
- getElementsByTagName
- getElementsByName
- querySelectorAll
- querySelector
- createElement
- write
